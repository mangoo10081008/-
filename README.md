# 🚀 Study Quest (스터디 퀘스트)
> **"공부를 게임처럼, 성장을 눈으로 확인하는 AI 학습 플랫폼"**

## 💡 프로젝트 개요
공부하기 싫은 학생들의 마음을 가장 잘 아는 고등학생 창업가가 만드는 **에듀테크 서비스**입니다. 
지루한 문제 풀이에 게임의 '퀘스트'와 '보상' 개념을 도입하여 학습 동기를 극대화합니다.

## 🔥 핵심 기능 (Core Features)
1. **AI 문제 분석 엔진**: Google Gemini 1.5 Pro를 연동하여 사진 한 장으로 [정석 풀이]와 [야매 풀이]를 동시에 제공합니다.
2. **게이미피케이션**: 공부 시간에 비례하여 XP(경험치)를 얻고, 문제 난이도에 따라 Gold를 획득합니다.
3. **나만의 교재 보관함**: 시중의 교재를 검색하여 나만의 학습 리스트를 만들고 관리합니다.
4. **성취도 시각화**: 내가 얼마나 공부했는지 실시간 대시보드로 확인합니다.

## 🛠 Tech Stack (기술 스택)
- **Frontend**: React.js, Vite
- **AI**: Google Gemini API
- **Version Control**: Git & GitHub (매일 잔디 심기를 통해 성실함 증명 중)

## 📈 창업가로서의 다짐
- **매일 커밋**: 하루도 빠짐없이 코드를 개선하고 기록합니다.
- **사용자 중심**: 실제 친구들의 피드백을 받아 서비스를 고도화합니다.
- **자립**: 1인 창업가로서 기술력과 비즈니스 마인드를 동시에 갖춘 인재로 성장하겠습니다.
